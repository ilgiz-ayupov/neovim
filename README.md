## Настройки редактора

### Нумерация строк

- Включена обычная и относительная нумерация строк
- Удобно для навигации и vim-движений (`5j`, `3k` и т.д.)

---

### Вкладки и отступы

- Размер таба: **2 пробела**
- Отступы: **2 пробела**
- Табы автоматически преобразуются в пробелы
- Новый ряд наследует отступ текущей строки

Подходит для большинства JS / TS / Lua / Web-проектов и совпадает с настройками Prettier.

---

### Отображение

- Перенос строк отключён
- Подсвечивается строка с курсором
- Колонка знаков (`signcolumn`) отображается всегда, без сдвига текста

---

### Поиск

- Поиск без учёта регистра по умолчанию
- Если в запросе есть заглавные буквы — поиск становится регистрозависимым

---

### Цвета и тема

- Включена поддержка 24-битного цвета (`termguicolors`)
- Используется тёмный вариант цветовых схем
- Требуется терминал с поддержкой true color

---

### Буфер обмена

- Используется системный буфер обмена (`clipboard=unnamedplus`)
- Копирование и вставка работают между Neovim и системой

---

### Окна и сплиты

- Вертикальные сплиты открываются справа
- Горизонтальные сплиты открываются снизу

---

### Прочее

- Swap-файлы отключены
- Backspace работает предсказуемо: на отступах, конце строки и в начале insert-режима
- Netrw использует древовидный стиль отображения файлов

## Общие сочетания клавиш

### Режимы

| Клавиши | Описание |
|--------|----------|
| `jk` | Перейти в Normal mode |

---

### Поиск

| Клавиши | Описание |
|--------|----------|
| `Esc` | Очистить подсветку результатов поиска |

---

### Управление окнами

| Клавиши | Описание |
|--------|----------|
| `<leader>sv` | Разделить окно вертикально |
| `<leader>sh` | Разделить окно горизонтально |
| `<leader>se` | Сделать окна одинакового размера |
| `<leader>sx` | Закрыть текущее окно |

---

### Управление вкладками

| Клавиши | Описание |
|--------|----------|
| `<leader>to` | Открыть новую вкладку |
| `<leader>tx` | Закрыть текущую вкладку |
| `<leader>tn` | Перейти к следующей вкладке |
| `<leader>tp` | Перейти к предыдущей вкладке |
| `<leader>tf` | Открыть текущий буфер в новой вкладке |

---

### Примечания

- `<leader>` установлен в `Space`
- Все сочетания определены через `vim.keymap.set`
- Конфигурация ориентирована на минимальные движения рук

## LSP сочетания клавиш

### Навигация по коду

| Клавиши | Описание |
|--------|----------|
| `gd` | Перейти к определению |
| `gD` | Перейти к объявлению |
| `gR` | Показать все ссылки (references) |
| `gi` | Показать реализации |
| `gt` | Показать определения типов |

---

### Действия с кодом

| Клавиши | Описание |
|--------|----------|
| `<leader>ca` | Действия кода (code actions), работает и в visual-режиме |
| `<leader>rn` | Умное переименование символа |

---

### Диагностика

| Клавиши | Описание |
|--------|----------|
| `<leader>D` | Диагностика текущего файла (Telescope) |
| `<leader>d` | Диагностика текущей строки |
| `[d` | Перейти к предыдущей ошибке/предупреждению |
| `]d` | Перейти к следующей ошибке/предупреждению |

---

### Документация

| Клавиши | Описание |
|--------|----------|
| `K` | Показать документацию под курсором |

---

### LSP

| Клавиши | Описание |
|--------|----------|
| `<leader>rs` | Перезапустить LSP |

---

### Примечания

- Все биндинги активны только при подключённом LSP (`LspAttach`)
- Навигация и диагностика используют **Telescope**
- Все биндинги являются **buffer-local**
