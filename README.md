## Настройки редактора

### Нумерация строк

- Включена обычная и относительная нумерация строк
- Удобно для навигации и vim-движений (`5j`, `3k` и т.д.)

---

### Вкладки и отступы

- Размер таба: **2 пробела**
- Отступы: **2 пробела**
- Табы автоматически преобразуются в пробелы
- Новый ряд наследует отступ текущей строки

Подходит для большинства JS / TS / Lua / Web-проектов и совпадает с настройками Prettier.

---

### Отображение

- Перенос строк отключён
- Подсвечивается строка с курсором
- Колонка знаков (`signcolumn`) отображается всегда, без сдвига текста

---

### Поиск

- Поиск без учёта регистра по умолчанию
- Если в запросе есть заглавные буквы — поиск становится регистрозависимым

---

### Цвета и тема

- Включена поддержка 24-битного цвета (`termguicolors`)
- Используется тёмный вариант цветовых схем
- Требуется терминал с поддержкой true color

---

### Буфер обмена

- Используется системный буфер обмена (`clipboard=unnamedplus`)
- Копирование и вставка работают между Neovim и системой

---

### Окна и сплиты

- Вертикальные сплиты открываются справа
- Горизонтальные сплиты открываются снизу

---

### Прочее

- Swap-файлы отключены
- Backspace работает предсказуемо: на отступах, конце строки и в начале insert-режима
- Netrw использует древовидный стиль отображения файлов

## Форматирование кода

### Conform (Auto-formatter)

- **Автоматическое форматирование** при сохранении файла
- Поддерживаемые инструменты:
  - Lua: `stylua`
  - Go: `gofumpt`
  - JavaScript/TypeScript: `prettier`
  - Web форматы (CSS, HTML, JSON, YAML, Markdown): `prettier`

## Дополнительные плагины

### Autopairs

- Автоматическое закрытие скобок `()`, `[]`, `{}`
- Интеграция с auto-completion для правильной работы с функциями

### Blink.cmp

- Современное автодополнение с поддержкой:
  - LSP
  - Snippets (friendly-snippets)
  - Path completion
  - Buffer completion

### Blink.pairs

- Подсветка парных скобок и символов

### Treesitter

- Синтаксический анализ для: Lua, Go, Bash, Dockerfile, Markdown, JSON, YAML, TOML
- Улучшенная подсветка синтаксиса
- Поддержка операторов как текстовых объектов

### Mason & Mason Tool Installer

- Автоматическая установка LSP серверов и форматтеров
- Установленные инструменты: lua_ls, gopls, stylua, gofumpt, prettierd

### Indent-blankline

- Визуальные линии отступа для лучшей читаемости кода
