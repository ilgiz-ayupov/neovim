# Инструкция по работе с lazygit
**lazygit** — это удобный терминальный интерфейс для git, который позволяет выполнять все основные операции с репозиторием через простое TUI-меню.

## Запуск
Откройте терминал в папке вашего git-репозитория и выполните:
```bash
lazygit
```

## Основные возможности
- **Статус**: Просмотр текущих изменений, неиндексированных и индексированных файлов.
- **Коммиты**: Просмотр истории коммитов, подробностей, diff, откат к коммиту.
- **Staging/unstaging**: Добавление/удаление файлов и отдельных изменений (hunks) в индекс.
- **Merge/Conflict**: Разрешение конфликтов с помощью встроенного редактора.
- **Push/Pull**: Отправка изменений на удалённый репозиторий и получение новых.
- **Работа с ветками**: Создание, удаление, переключение, слияние, rebase веток.
- **Работа с тегами**: Просмотр, создание, удаление тегов.
- **Интерактивный rebase**: Перестановка, squash, удаление коммитов.
- **Просмотр и восстановление stash**: Сохранение и возврат временных изменений.
- **Git blame**: Просмотр истории изменений по строкам файла.
- **Работа с remotes**: Добавление, удаление, fetch, push/pull с разных удалённых репозиториев.

## Основные клавиши
- `Tab` — Переключение между панелями (статус, коммиты, файлы и т.д.)
- `↑/↓` — Навигация по спискам
- `Enter` — Открыть подробности/выполнить действие
- `Space` — Stage/unstage файл или hunk
- `c` — Создать коммит
- `p` — Push
- `P` — Pull
- `b` — Открыть меню веток
- `m` — Слияние ветки (merge)
- `r` — Rebase
- `d` — Просмотр diff
- `s` — Stash
- `x` — Восстановить из stash
- `f` — Fetch
- `l` — Log (история)
- `e` — Редактировать конфликт вручную
- `q` — Выйти из текущего окна/панели
- `?` — Справка по горячим клавишам

## Разрешение конфликтов
- Перейдите к файлу с конфликтом (отмечен знаком ⚡).
- Нажмите `Enter` для просмотра diff.
- Используйте `e` для ручного редактирования или выберите нужную версию (ours/theirs/both).
- После разрешения — добавьте файл в индекс (`Space`).

## Советы
- Все действия можно выполнять мышкой (если поддерживается терминал).
- Для подробной справки нажмите ? в любой панели.
- lazygit не вносит изменений в git-конфиг, работает только с текущим репозиторием.
