# Инструкция по работе с lazydocker

**lazydocker** — это терминальный интерфейс для управления Docker-контейнерами, образами, томами и сетями через удобное TUI-меню.

---

## Запуск

Откройте терминал и выполните:

```
lazydocker
```

---

## Основные возможности

- Просмотр всех контейнеров, образов, томов и сетей
- Запуск, остановка, перезапуск, удаление контейнеров
- Просмотр логов контейнера в реальном времени
- Просмотр и управление процессами внутри контейнера
- Просмотр и удаление неиспользуемых образов, томов, сетей
- Управление переменными окружения контейнера
- Просмотр информации о ресурсах (CPU, память, сеть)
- Инспекция и удаление объектов Docker
- Быстрый доступ к Docker Compose (если есть docker-compose.yml)

---

## Основные клавиши

- `Tab` — Переключение между панелями (контейнеры, образы, тома, сети)
- `↑/↓` — Навигация по спискам
- `Enter` — Открыть подробности/меню объекта
- `Space` — Выполнить действие (старт/стоп/удалить и т.д.)
- `l` — Просмотр логов выбранного контейнера
- `e` — Открыть редактор переменных окружения
- `r` — Перезапустить контейнер
- `s` — Остановить контейнер
- `d` — Удалить объект (контейнер, образ, том, сеть)
- `c` — Открыть shell внутри контейнера (если поддерживается)
- `p` — Просмотр процессов внутри контейнера
- `x` — Принудительно удалить объект
- `f` — Обновить список объектов
- `q` — Выйти из текущей панели/закрыть окно
- `?` — Справка по горячим клавишам

---

## Советы

- Для подробной справки по действиям нажмите `?`
- lazydocker не изменяет ваши Docker-файлы, работает только с объектами Docker
- Можно использовать мышь (если поддерживается терминал)
- Все действия выполняются мгновенно, без необходимости писать длинные команды
